# Funnel Scripts & Copy Templates

Scripts for VSL (video sales letters), webinar openers, opt-in pages, and sales calls.

---

## VSL Script Framework (2–5 minutes)

```
[0:00–0:15] HOOK
"If you're a [audience] who [situation], pay attention for the next [X] minutes because I'm going to show you [promise]."

[0:15–0:45] PROBLEM
"Right now, you're probably dealing with [pain 1], [pain 2], and [pain 3]. And the reason this keeps happening is [root cause]."

[0:45–1:30] AGITATE
"And here's the thing — if nothing changes, [consequence]. I see this all the time with [audience]."

[1:30–2:30] CREDIBILITY
"My name is [name]. I [brief credibility statement]. I've helped [X clients] [result]."

[2:30–3:30] SOLUTION
"What I've discovered is a [framework name] that [core mechanism]. Instead of [old way], you [new way]."

[3:30–4:15] PROOF
"[Client name/type] used this and went from [before] to [after] in [timeframe]. [Another example]."

[4:15–5:00] CTA
"Here's what to do next: [clear single action]. Click the button below / fill in the form / book a call. [Urgency or reason to act now]."
```

## Opt-In Page Copy

### Above the Fold
```
HEADLINE: [Specific outcome] for [audience] — without [pain/sacrifice]

SUBHEADLINE: [What they'll get] + [timeframe or ease]

LEAD MAGNET NAME: "The [Audience] [Outcome] [Format]"
e.g., "The Agency Owner's 5-Day Lead Machine Blueprint"

CTA BUTTON: "Send Me The [Format] →"

MICRO-COPY (below button): No spam. Unsubscribe anytime.
```

### What You'll Learn Section
```
Inside this free [format], you'll discover:

✅ [Specific insight 1 — curiosity-driven]
✅ [Specific insight 2 — addresses objection]
✅ [Specific insight 3 — biggest pain point]
✅ [Bonus insight — surprising or contrarian]
```

## Sales Call Script

### Opening (2 min)
```
"Thanks for jumping on — before we get into things, I want to set some expectations.

This is a conversation, not a pitch. My job is to understand where you're at and figure out if we can help. If we can't, I'll tell you honestly.

Sound good? Great. So tell me — what made you book this call today?"
```

### Discovery (10–15 min)
```
Key questions:
1. "What does your [lead gen / content / sales process] look like right now?"
2. "Where's the biggest bottleneck for you?"
3. "What have you already tried? What happened?"
4. "If you could fix one thing in the next 90 days, what would it be?"
5. "What does success actually look like for you — what changes?"
6. "What's the cost of NOT solving this? Revenue, time, stress?"
```

### Transition to Offer
```
"Based on what you've shared, I think I can help. Can I walk you through how we typically approach this?"

[Explain your process/offer simply]

"Does that resonate with what you need?"
```

### Handling Objections

**"I need to think about it"**
```
"Totally get it — what specifically do you need to think through? Is it the investment, the timing, or something about the approach?"
[Address the real objection, then re-ask]
```

**"It's too expensive"**
```
"I hear you. Can I ask — what's it costing you right now to NOT have this solved? [Wait for answer] So the real question is whether [price] is worth [outcome]. Does that frame it differently?"
```

**"I need to speak to my partner/team"**
```
"Of course. If they were on this call, what do you think their main questions would be? Let me help you answer those now."
```

### Close
```
"Based on everything we've discussed, it sounds like [summary of their situation and goals]. [Offer] is designed exactly for that.

Are you ready to move forward, or is there anything else you need from me to make that decision?"

[If yes] → "Perfect. Let's get you started. I'll send over the agreement and next steps in the next 10 minutes."

[If hesitant] → "What would need to be true for you to feel confident saying yes today?"
```

Check your inbox — your [lead magnet] is on its way.
