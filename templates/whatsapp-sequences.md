# WhatsApp Follow-Up Sequences

Copy, personalise, and load these into your automation tool (GHL, Twilio, n8n).

---

## Sequence 1: Lead Follow-Up (5-Day)

### Day 0 — Instant Response (within 60 seconds)
```
Hey {{name}}! 👋

Got your request — thanks for reaching out.

I'll personally get back to you shortly with everything you need.

Reply YES to confirm you received this.
```

### Day 1 — Value Add
```
Hey {{name}}, it's [Your Name] again.

Quick one — I put together a short guide that covers exactly what most [target audience] get wrong when it comes to [problem].

Want me to send it over? Just reply YES.
```

### Day 2 — Social Proof
```
{{name}}, just wanted to share something.

[Client name] came to us 3 months ago with the exact same situation you're in.

Here's what happened: [1-2 sentence result]

Worth a quick chat? I have 2 spots open this week.
```

### Day 3 — Direct Ask
```
Hey {{name}} — still thinking about [their problem/goal]?

I work with [target audience] to [outcome] without [pain/sacrifice].

If that sounds relevant, let's hop on a 20-min call this week. No pitch — just clarity.

Reply CALL and I'll send you a link.
```

### Day 4 — Objection Handle
```
{{name}}, I get it — timing might not be right.

But here's something I've seen with most [audience]:

Waiting usually costs more than acting. [Brief stat or story]

Still here if you need anything. Reply anytime. 🙏
```

### Day 5 — Final Touchpoint
```
Last message from me, {{name}}.

If you ever want to explore [outcome], just reply to this message.

No pressure — just wanted to make sure I didn't drop the ball on my end.

Wishing you well either way 🙏
```

## Sequence 2: Booking Confirmation + Pre-Call

### Booking Confirmed
```
{{name}}, you're booked! ✅

📅 {{call_date}} at {{call_time}}
📞 {{zoom_or_call_link}}

To make our time together as valuable as possible, come ready to share:
1. Your biggest challenge right now
2. What you've already tried
3. What success looks like for you

See you then!
```

### 24h Reminder
```
Hey {{name}} — reminder that we're talking tomorrow!

📅 {{call_date}} at {{call_time}}
📞 {{zoom_or_call_link}}

Any questions before then, just reply here.
```

### 1h Reminder
```
{{name}}, we're on in 1 hour! ⏰

Here's your link: {{zoom_or_call_link}}

See you soon 🙌
```

### Post-Call Follow-Up
```
{{name}}, great talking with you today!

As discussed, here's what I'm sending over:
[List what you promised]

I'll follow up by [date]. Any questions, reply here.
```

## Sequence 3: Re-Engagement (Cold Leads)

### Message 1
```
Hey {{name}} — it's been a while since we last connected.

A lot has changed with [your offer/approach] and I wanted to reach out personally.

Are you still working on [goal/problem]?
```

### Message 2 (if no reply after 3 days)
```
{{name}}, dropping this here in case my last message got buried.

I've been working with [audience] on [outcome] and seeing some strong results.

Worth 15 minutes to see if there's a fit?
```

## Sequence 4: Post-Purchase / Onboarding

### Welcome
```
Welcome to [Program/Service], {{name}}! 🎉

You've made a great decision. Here's what happens next:

1. [Step 1]
2. [Step 2]
3. [Step 3]

Your onboarding call is booked for [date]. Any questions, I'm right here.
```

### Day 3 Check-In
```
Hey {{name}} — how are you finding things so far?

Any wins to celebrate? Any questions? Just reply here — I'm paying attention.
```
