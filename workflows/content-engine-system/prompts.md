# AI Prompts — Content Engine System

Copy and adapt these prompts directly into your workflow or use them manually with Claude/ChatGPT.

---

## 🎣 Hook Generator

```
Generate 10 scroll-stopping hooks for [target audience] struggling with [problem].

Vary the styles:
- Direct question
- Bold/controversial statement  
- Surprising statistic
- "If you..." opener
- Story/personal experience
- Myth-busting
- "The truth about..."
- Number-led ("3 reasons why...")
- Relatability ("You know that feeling when...")
- Urgency-based

Format as numbered list. Keep each hook under 15 words.
```

**Example output for coaches:**
1. Why most coaches fail in year one (it's not what you think)
2. Your potential clients aren't saying no to you. They're saying no to your offer.
3. 87% of coaches undercharge by at least 40%. Are you one of them?
4. If you're posting daily and still not getting clients, read this.
5. I went from $0 to $10k/month without a big audience. Here's the system.

## 🎬 Reel Script Generator

```
Write a 30-second Instagram Reel script using the Problem-Agitate-Solution framework.

Details:
- Topic: [topic]
- Target Audience: [audience]
- Main Pain Point: [pain point]
- Solution/Offer: [your solution]
- CTA: [what you want them to do]
- Tone: [conversational / authoritative / energetic]

Format exactly like this:
HOOK (0-5s): [opening line — must stop the scroll]
PROBLEM (5-13s): [name the pain clearly]
AGITATE (13-20s): [make the pain feel real and urgent]
SOLUTION (20-27s): [introduce your solution simply]
CTA (27-30s): [single clear action]
```

## 📝 LinkedIn Post Generator

```
Write a LinkedIn post about [topic] for [target audience].

Requirements:
- Start with a pattern-interrupt first line (no "I'm excited to share...")
- Use short paragraphs (1-3 lines max)
- Include white space for readability
- Add a personal insight or contrarian take
- End with a question that invites comments
- Tone: [professional but conversational / direct / storytelling]
- Max 1,300 characters

Topic context: [brief description of what you want to say]
```

---

## 🐦 Twitter/X Thread Generator

```
Write a 7-tweet thread about [topic] for [audience].

Format:
Tweet 1 (Hook): Bold claim or surprising insight. Must make people want to read on. End with "Thread 🧵"
Tweets 2-6: One insight per tweet. Short. Punchy. Actionable.
Tweet 7 (CTA): Summarise the value + tell them what to do next.

Rules:
- Each tweet max 280 characters
- No fluff — every word earns its place
- Use numbers where possible (specificity = credibility)
- Topic: [topic]
```

## 📸 Instagram Caption Generator

```
Write an Instagram caption for a post about [topic].

Structure:
- Line 1: Hook (makes them tap "more")
- Lines 2-4: Value / story / insight
- Line 5: CTA (comment, DM, save, share)
- Hashtags: 5-10 relevant hashtags at the end

Audience: [target audience]
Tone: [tone]
Post context: [what the image/video shows]
```

## 📧 Email Newsletter Generator

```
Write a short email newsletter (300-400 words) about [topic].

Structure:
- Subject line: [curiosity-driven, max 8 words]
- Preview text: [complements subject line]
- Opening: personal hook or story
- Body: one useful insight or framework
- CTA: one clear next step (no multiple CTAs)

Audience: [audience]
Tone: [conversational, like writing to a friend]
Topic: [topic]
```

## 🔄 Content Repurposing Prompt

```
I have this piece of content:
[paste your content]

Repurpose it into:
1. A 30-second Reel script
2. A LinkedIn post
3. A Twitter/X thread (5 tweets)
4. An email subject line + preview text
5. 3 Instagram caption options

Keep the core message the same. Adapt tone and format for each platform.
```
